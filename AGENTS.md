# Overview

This repository contains the source to build a Jekyll website that is deployed via GitHub actions on GitHub Pages. Beyond Jekyll, the site mostly relies on Bootstrap 5 for styling and responsiveness. Design and content should be kept separate as much as possible. Layouts should be reused and nested layouts are encouraged.

The website represents an academic research group in theoretical machine learning. Thus it needs to look professional and modern. Content should be easy to navigate.
